Weather Application

This weather application allows you to access the current weather conditions in your location, and in the cities you searched, as well as the forecast for the next six days. Depending on your preference, you can switch between light and dark themes. To create this application, I used HTML, CSS and Javascript. It also consults an API called Open Weather to collect the weather data, and I used Bootstrap and React as a framework to create this application.

Dictionary Application

One of my recent projects was creating a dictionary application using React. This application allows users to easily consult a dictionary API for definitions and synonyms of words. Additionally, I incorporated an image API to provide users with visual representations of their searched words. The user interface is simple and intuitive, making it easy for anyone to use.

Tourism Landing Page

This is a simple tourism landing page about Rio de Janeiro. My focus was to work on my responsive development skills, and I used HTML and CSS to develop the page. The website is designed to be easily accessible on different devices, ensuring a smooth user experience regardless of the screen size. The page features stunning images of Rio de Janeiro and provides useful information for anyone planning a trip to this beautiful city.

World Clock

This is a world clock application that has been created using HTML, CSS and Javascript. It consults an API called moment.js, in this application, you can check your current day and time and also from other important capitals around the world.

The Chosen landing page

I created a landing page for the show "The Chosen" as a final project for my Shecodes basics course. I used HTML, CSS, and Javascript to create the landing page, which included a link button for viewing episodes of the show in their official website, descriptions about the show's cast and project mission.
